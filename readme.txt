A WordPress multi-site plugin that scans for viruses using ClamAV.

== Description ==

This is a WordPress plugin that utilizes the power of ClamAV to scan files and folders in your WordPress multi-site installation. This plugin provides an admin interface to initiate the scans and view the results.

== Installation ==

1. Upload the `clamav-multi-site-scanner` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to 'Tools' -> 'ClamAV Scanner' to configure and run scans

== Frequently Asked Questions ==

= Is ClamAV required to be installed on the server? =

Yes, ClamAV needs to be installed on the server where WordPress is running.
